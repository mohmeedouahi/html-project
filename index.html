Guide GitHub : Qu'est-ce que GitHub, Comment Créer un Compte et Importer des Projets Locaux

Qu'est-ce que GitHub ?

GitHub est une plateforme de développement collaborative basée sur Git. Git est un système de contrôle de version qui permet aux développeurs de suivre l'évolution de leur code source.

Comment Créer un Compte GitHub :

  1. Accédez à la page d'accueil de GitHub : https://github.com/
  2. Cliquez sur "Sign up for GitHub" et remplissez le formulaire avec votre nom d'utilisateur, votre adresse e-mail et un mot de passe.
  3. Vous recevrez un e-mail de vérification. Cliquez sur le lien dans l'e-mail pour vérifier votre adresse e-mail.
  4. Configurez votre compte en ajoutant un profil et en sélectionnant un plan (GitHub offre des comptes gratuits et des comptes payants).

Importation des Projets Locaux sur GitHub :

  1. Créez un nouveau répertoire (repository) sur GitHub en cliquant sur "New". Donnez-lui un nom et une description.
  2. Sur votre ordinateur, ouvrez un terminal et allez dans le répertoire de votre projet local.
  3. Tapez git init pour initialiser un nouveau dépôt Git local.
  4. Tapez git add . pour ajouter tous les fichiers au suivi de version.
  5. Tapez git commit -m "Premier Commit" pour valider les fichiers.
  6. Copiez l'URL du répertoire GitHub que vous venez de créer.
  7. Tapez git remote add origin pour ajouter le répertoire GitHub comme origine.
  8. Tapez git push -u origin master pour pousser votre code local sur GitHub.

GitHub est une ressource inestimable pour les développeurs, offrant des outils puissants pour la collaboration et la gestion du code.
